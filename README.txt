There are 2 versions of the program. (2 versions of javascript file)

-- The 1st version: meets all requirements of the project and simply log
results to the console

-- The 2nd version: beyond the requiremnts of the project. It display the
results in the actual DOM and let users interact with program using the mouse.


# By default the index.html file will link to the app.js file for the 1st version.

To use the 2nd version, simply unlink the app.js and link index.html file to the
app_dom.js file
